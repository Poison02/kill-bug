# 项目简介
kill-bug是一个易于使用的平台，专为程序员解决日常开发中遇到的问题而设计。它提供搜索问题、发布问题、回答问题、发布悬赏任务、即时通信、协同编辑等功能。
灵感来源于这个网站：[openask](https://openask.me/home)，但是功能却要小得多。
